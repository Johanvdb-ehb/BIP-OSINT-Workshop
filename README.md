# BIP OSINT Workshop

Welcome to the OSINT (Open Source Intelligence) workshop!  
This repository contains all materials used during the session.

## Workshop Goals
By the end of this session, you will be able to:
- Understand what OSINT is and why it matters in ethical hacking
- Use Google Dorking to find exposed information
- Extract metadata and frames from media for investigation
- Use Shodan to discover exposed devices (e.g. webcams)
- Use Maltego to map online footprints (people / infrastructure)
- Think ethically and responsibly about public data
